# Glossary

Use this file to anchor the project vocabulary before feature-specific terms spread across contracts, KB pages, and runbooks.

| Term | Meaning |
|---|---|
| product artifact | a project-local deliverable such as a dataset, service, model, or report |
| operational evidence | logs, metrics, alerts, reports, or approval artifacts used for diagnosis and audit |
| control plane | the set of readiness, security, cost, token, and governance checks used before ship |
| sentinel | the supervised multi-agent monitoring layer for runtime, security, quality, and operational intelligence |
| quarantine | the safe containment path used when uncertainty or risk is high enough to block autonomous continuation |
