# Privacy Policy (Short)

AgentCodex is designed to install and maintain a project-local scaffold inside a repository.

It may process:

- repository file paths and directory structure
- repository-local configuration files
- prompts or instructions given by the user in Codex
- scaffold files written into the target repository

AgentCodex is designed around project-local state and is not intended to operate as an advertising, tracking, or profile-building system.

If AgentCodex is distributed through Codex, a package registry, or another platform, that platform may also collect operational metadata under its own policies.

Users and publishers remain responsible for reviewing repository contents, avoiding unnecessary secrets in prompts, and applying any compliance or retention controls required for their environment.

For the full version, see `docs/PRIVACY.md`.
